> # Switch Statement In Java:
 It executes one statement from multiple conditions.
    
__Syntax:__

```java
switch(expression){

    case value01:
    //Statement01
    break;

    case value02:
    //Statement02
    break;

    ...

    case valueN:
    //StatementN
    break;

    default:
    //code to be executed if all cases arenot matched.

}
```
